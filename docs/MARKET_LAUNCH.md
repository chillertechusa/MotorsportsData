# Motorsports Data — Market Launch Playbook

## Go-To-Market Strategy

### Phase 1: Coach Validation (Weeks 1-2)
**Goal:** Book 5-10 demo calls, gather feature feedback, validate pricing

**Actions:**
1. Reach out to 20 coaches via LinkedIn + email
   - Message template: `"We built a data platform for elite motocross coaches. 30-min demo of readiness scores + accountability tracking. Interested?"`
   - Target: Coaches with 5+ professional riders

2. Demo booking flow
   - Direct coaches to `motorsportsdata.io/demo-booking`
   - Offer live 30-min demo or asynchronous 5-min video walkthrough

3. During demo, gather feedback on:
   - Readiness score accuracy + usefulness
   - Template encryption + IP protection
   - Multi-rider comparison value
   - Pricing sensitivity ($49 vs $99 vs $199)
   - Must-have features before signup

4. Track feedback in shared doc:
   ```
   Coach Name | Company | Fit Score (1-10) | Main Need | Willing to Pay | Blockers
   ```

**Success metric:** 3+ coaches willing to pilot (even if free)

---

### Phase 2: Pilot Cohort (Weeks 3-4)
**Goal:** Get 3-5 coaches using the platform, real data flowing

**Actions:**
1. Onboard pilots via `/onboarding` flow
2. Load demo data (2-week training progression + race results)
3. Set up weekly check-in calls (30 min each)
4. Gather usage metrics:
   - Time spent on platform
   - Most-used features
   - Missing features or bugs
   - Net promoter score (NPS)

**Success metric:** All pilots logging in 3+ times/week, asking feature questions

---

### Phase 3: Feature Refinement (Weeks 5-6)
**Goal:** Build features pilots are asking for

**Likely requests:**
- Live telemetry streaming (already built, just needs wiring)
- Coach-to-rider messaging
- PDF export of reports
- Mobile app or PWA
- Integration with specific devices (Garmin, Apple Watch)

**Action:** Prioritize by pilot feedback + effort

---

### Phase 4: Pricing & Scaling (Weeks 7+)
**Goal:** Move from pilots to paying customers

**Pricing model (validated via pilots):**
- Rookie tier: $0 (free trial, limited riders)
- Race Team tier: $49/mo (all features, 1-5 riders)
- Factory Rig tier: $199/mo (multi-rider, API access, support)

**Sales motion:**
1. Pilots graduate to paid (or churn + learn why)
2. Run Google Ads: "Coach SaaS", "Motocross data analytics", "Rider readiness tracking"
3. Target keywords: "athlete readiness", "periodization software", "motocross coaching tools"
4. Measure CAC (customer acquisition cost) vs LTV (lifetime value)

## Metrics to Track

### Engagement
- Demo booking rate (% of emails that book)
- Demo-to-pilot conversion (% of demos that sign up)
- Feature usage (which views are most visited)
- Session duration (how long coaches spend on platform)

### Business
- CAC (cost per acquired customer)
- LTV (lifetime value per customer)
- MRR (monthly recurring revenue)
- Churn rate (% of pilots who drop out)

### Product
- NPS (net promoter score from pilots)
- Feature requests (prioritize by frequency)
- Bug reports (track severity)
- Support tickets (time to resolution)
